
# Exercism C# Solutions
Este repositório contém minhas **soluções em C#** para os exercícios da plataforma **[Exercism](https://exercism.org)**, utilizando o track de **C#**.

Os exercícios servem como prática para fortalecer a **lógica de programação**, aprofundar o **conhecimento da linguagem C#** e desenvolver habilidades com o **ecossistema .NET**.

## Sobre o Exercism
O **Exercism** é uma plataforma gratuita focada no aprendizado prático de programação por meio de exercícios com **testes automatizados**.  
Cada _track_ (como C#, Python, Java, etc.) apresenta desafios progressivos que ajudam a consolidar conceitos, boas práticas e padrões de código.

## Ferramentas Recomendadas
Para acompanhar ou executar os exercícios localmente, recomenda-se:
-   **.NET SDK**
-   **Visual Studio** ou **Visual Studio Code**
-   **CLI do Exercism**
    
## Objetivo do Repositório
-   Manter um histórico do meu aprendizado em C#
-   Reforçar os fundamentos da linguagem
-   Facilitar revisões e consultas futuras

## Observações
Algumas soluções incluem comentários explicativos com foco em aprendizado.
Ao longo do tempo, soluções podem ser refatoradas conforme meu conhecimento evolui.
